Current guardrails

What stays unchanged in this build
The mobile refactor keeps the static deployment, but the Busses gate now uses real auth.
  • Public pages remain public and keep their current URLs.
  • Protected pages require a signed-in Supabase session from an authorized admin email.
  • Authorized admin emails are linked to player records and managed from Busses Squad.
  • Supabase Auth email templates should send the 6-digit OTP code, not a magic link.
  • `season_id` stays in the URL flow for season-aware pages.
  • Supabase browser data loading remains in the existing static HTML, CSS, and JS structure.
